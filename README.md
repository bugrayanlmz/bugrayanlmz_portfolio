## Installation

Follow these steps to run the project:

### 1. Load dependencies

```bash
npm install
```

### 2. Launch the development server

```bash
npm start
```

The application will be available at [http://localhost:3000](http://localhost:3000).

### 3. Creating a production build

```bash
npm run build
```
